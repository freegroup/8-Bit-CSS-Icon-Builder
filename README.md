# 🎮 8-Bit CSS Icon Builder

**[▶ Try it live!](https://freegroup.github.io/8-Bit-CSS-Icon-Builder/)**

Paint pixel art icons and export them as **pure CSS** – no images, just `box-shadow` magic! ✨

## Features

- 🎨 16 retro colors (C64/NES palette)
- 📐 8x8 or 16x16 grid
- ✏️ Pen, Line, Rectangle, Circle tools
- 👀 Live preview at 1x, 2x, 4x
- 📋 CSS export & import

## Usage

1. Pick a color → Paint pixels → CSS Preview → Copy!

```html
<div class="my-icon"></div>
```

## Install

Just open `index.html`. Done.
